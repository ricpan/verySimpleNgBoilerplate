The very simple ng boilerplate

About

The goal is to have a basic structure to get started with AngularJS apps. Enjoy!

What is it?

Kickstart any AngularJS application without repeating yourself on basic tasks giving you freedom to extend the way you want/need.

How to use it?

Clone this repo, delete .git and go make some apps.

Do you need a simple http server?

A simple zero-configuration command-line http server: http-server
